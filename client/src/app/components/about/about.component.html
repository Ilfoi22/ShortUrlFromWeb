URL Shortener Algorithm

    Generate Unique Code:

  • Utilizes a service (UrlShorteningCreateCodeService) to generate a unique code for each URL.
  • A random alphanumeric string of a specified length is generated.
  • The uniqueness of the code is ensured by checking against the database.

    Create Short URL:

  • The CreateShortUrlAsync method in the ShortenUrlController handles the creation of the short URL.
  • Validates the provided long URL.
  • Generates a unique code using the UrlShorteningCreateCodeService.
  • Constructs the short URL using the generated code and the current request's scheme and host.
  • Stores the long URL, short code, and additional metadata (creation date, user ID) in the database.

    Retrieve Short URL:

  • The GetShortUrlAsync method in the ShortenUrlController retrieves the original long URL based on the short code.
  • If the code is found in the database, it redirects to the original long URL.

    Delete Short URL:

  • The DeleteShortUrlAsync method in the ShortenUrlController handles the deletion of a shortened URL.
  • Checks if the user is an admin or the creator of the short URL before allowing deletion.
  • Removes the entry from the database.
This algorithm ensures the creation of unique short URLs, supports retrieval and redirection to the original URLs, and provides administrative capabilities for URL management.
